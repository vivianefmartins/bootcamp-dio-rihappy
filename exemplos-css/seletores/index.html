Bem-vindo!

No CSS nós possuímos os seletores por ID

  • Seletor por Tipo: Busca elementos por uma tag HTML específica.
  • Seletor de ID (#): Busca elementos através do atributo "id".
  • Seletor por Class (.): Busca elementos através do atributo "class".
  • Seletor Universal (*): Seleciona todos os elementos HTML.
  • Seletro de Atributo ([atrib]): Seleciona elementos que possuem um atributo específico em nosso documento HTML e conseguimos também buscar atributos com um valor específico.
